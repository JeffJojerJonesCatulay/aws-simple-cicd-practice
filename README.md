## aws-simple-cicd-practice

A simple static web application that demonstrates a basic CI/CD pipeline using AWS services. The project displays data from a dummy JSON file (e.g., name and catchphrase) and automatically deploys updates to an Amazon S3-hosted website through AWS CodePipeline and CodeBuild.

This project focuses on understanding how source code changes flow through a CI/CD pipeline—from GitHub to build processing and finally to deployment—using a minimal, cost-efficient setup.

### Tech Stack

* HTML, CSS, JavaScript
* AWS S3 (static website hosting)
* AWS CodePipeline (CI/CD orchestration)
* AWS CodeBuild (build automation)

### Key Features

* Fetches and displays data from a static JSON file
* Automated deployment on every push to the repository
* Simple and beginner-friendly CI/CD pipeline setup

### Purpose

This repository is intended for learning and practicing AWS CI/CD concepts using a lightweight static website, making it easy to understand the integration between services without backend complexity.
